    Node.js support

    All Node.js versions that are above or equal the release 14.0.0 (Fermium) are supported by Kipper.

    OS-wise Kipper only actively supports Windows 10/11 and Linux, even if Node.js supports more operating systems. (Therefore, Kipper will likely still work on more systems than listed here, but there is no guarantee.)
