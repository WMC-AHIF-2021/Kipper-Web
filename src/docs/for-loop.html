  • For-Loop

    For-loops are statements with a loop-expression, a loop condition and a statement body that are executed as long as a specified condition is met. For-loops have the unique attribute, unlike while-loops and do-while-loops, of having an executable LOOP_EXPRESSION, which is evaluated at the end of every loop cycle. This LOOP_EXPRESSION can be used very nicely to do repeating tasks at the end of a loop cycle, like calling a function or increasing a counter.

    Syntax

    for (INIT_EXPRESSION (OPTIONAL); CONDITION (OPTIONAL); LOOP_EXPRESSIONS (OPTIONAL)...) STATEMENT;

    The execution of a for-loop follows this schema:

    • Evaluate INIT_EXPRESSION, if it exists (Only the first time).
    • Check CONDITION, if it exists, before running the STATEMENT. If CONDITION is False, then the loop will be stopped!
    • Run STATEMENT if CONDITION was true.
    • Evaluate LOOP_EXPRESSION after finishing the execution of STATEMENT, and the loop was not stopped using return; or break;.

    Examples

    // ✓ Simple for-loop with an execution counter
for (var i: num = 1; i < 10; i++) {
  call print(f"Running for the {i}. time!");
}

// ✓ Simple for-loop with an execution counter
var j: num = 0;
for (var i: num = 1; i < 10; i++, j++) {
  call print(f"i = {i}");
}
call print(f"Additional variable j = {j}"); // -> Additional variable j = 10

// X Infinite loop - Avoid this, as it can result in freezing of your program without proper forced stopping
for ( ; ; ) {
  call print(f"Running for the {i}. time!");
}
