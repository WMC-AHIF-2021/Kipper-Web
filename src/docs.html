Kipper v0.1 - Documentation

Documentation and Base-Reference for the Kipper language

Our Goals:

  • Provide simple, straight-forward and useful explanations on the programming language Kipper.
  • Provide an interactive code-editor (Playground) to try out and use Kipper online.
