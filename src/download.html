Installation Requirements

Kipper runs on native TypeScript + Node.js, meaning you will need to install both of them. The links to their download sites are down below:

Node.js TypeScript

Kipper runs on all major platforms, including macOS, Windows and most linux distributions. Kipper also only requires a small amount of other modules to run properly, which will also be automatically handled by npm, meaning you will not have to download any other requirements other than Node.JS.

Npm Download

The recommend and easiest way to use Kipper, is to directly go to the npm

Install the npm module

The Kipper compiler may be installed using the npm npm package manager, meaning you can easily install it inside your terminal.

For simply trying out Kipper the following command can be used, which will globally install it and add the command to the path making it available for the entire nodejs 

npm install -g kipper

Important

Note though, that if possible avoid installing Kipper globally, as it can cause issues with locally installed versions (project dependencies). For a localised / project-dependent kipper installation add kipper to your package.json.

To add kipper to your project development dependencies use --save, to add it to the development dependencies use --save-dev 

npm install kipper --save/--save-dev

Run the Compiler

To run the Kipper Compiler, use the following command in the Console: 

kipper
