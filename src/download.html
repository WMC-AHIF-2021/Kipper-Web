Npm Download

The main source and most secure option to download Kipper is to directly download the source code from the npm package manager itself. For a guide on that, please go to the Install Guide.

Download Source Code

To download the latest stable source code of Kipper, download the zip or tar.gz with the links below:

  • Github Source Code (zip)
  • Github Source Code (tar.gz)

Downloading the GitHub Repository using Git

Git is the preferred way to use the source code of Kipper, as it allows pulling recent changes and updates directly into a directory. This is though recommended only for developers and people, who have an understanding of how git works and how to properly use it.

The url to the repository: https://github.com/Luna-Klatzer/Kipper.git
