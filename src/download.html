Guide to clone the GitRepo:

1. Open a Console-Interface

2. Switch to the Folder you would like to clone the Repo in

3. Type git clone https://github.com/Luna-Klatzer/Kipper.git in the Console


If everything is working git should start downloading the Repo to your specified Folder
