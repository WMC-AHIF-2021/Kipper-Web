Install Guide for Kipper

If you don´t want to install it, use our Playground

Pre-Setup

At first you need to install TypeScript and Node.js.

The Downloadlinks are below:

TypeScript

Node.js


Install dependencies

Whenever package.json is changed, you ned to run the following command:

npm install


Important:

Make sure to run the following Command in the Root Folder of your Project,

otherwise they wont´t work


Run the Application

To Run an Application use the following command in the Console:

npm start


Build your Project

To Build the Project use the following command:

npm run build
