Pre-Setup

At first you need to install TypeScript and Node.js. The download links are below:

TypeScript Node.js

Install the npm module

The package is hosted on the npm package manager, meaning you can easily fetch it using (This will install it as a global package and make it available in your terminal per default): 

npm install -g kipper

Important

If possible avoid installing Kipper globally, as it can cause issues with locally installed versions, even if it can be handy and easier. For a local TS installation add kipper to your 'package.json'.

Run the Compiler

To run the Kipper Compiler, use the following command in the Console: 

kipper
