Dependencies

Kipper runs native TypeScript + Node.js, meaning you will need to install both of them. The links to their download sites are down below:

Node.js TypeScript

The npm dependencies of Kipper will be automatically on installation added to the folder node_modules, meaning these won't have to be manually added.

Install the npm module

The Kipper-Compiler is hosted and kept up-to-date to the latest GitHub releases on the npm package manager, meaning you can easily install it using the command line.

For simply trying out Kipper the following command can be used, which will globally install it and add the command to the path making it available outside your project as well: 

npm install -g kipper

Important

Note though, that if possible avoid installing Kipper globally, as it can cause issues with locally installed versions (project dependencies). For a localised / project-dependent TS installation add kipper to your package.json.

To temporarily install the package kipper for trying out the kipper language you may do the following: 

npm install typescript --save-dev

Run the Compiler

To run the Kipper Compiler, use the following command in the Console: 

kipper
