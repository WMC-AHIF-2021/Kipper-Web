Dependencies

Kipper runs native TypeScript + Node.js, meaning you will need to install both of them. The links to their download sites are down below:

Node.js TypeScript

The npm dependencies of Kipper will be automatically on installation added to the folder node_modules, meaning these won't have to be manually added.

Install the npm module

The Kipper-Compiler is hosted and kept up-to-date on the node package manager (npm), meaning you can easily install it from there using the following command: 

npm install -g kipper

This will install the latest kipper version globally and make it available in the command line using kipper

Important

Note though, that if possible avoid installing Kipper globally, as it can cause issues with locally installed versions (project dependencies). For a localised / project-dependent TS installation add kipper to your project package.json file.

Run the Compiler

To run the Kipper Compiler, use the following command in the Console: 

kipper
